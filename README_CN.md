#FM（Financial Manager）

## 简介

* 个人财务管理的站点。

## 安装

### 安装环境
* apache/nginx + php + mysql

### 数据库表结构
* fm/db.sql

### 数据库配置
* fm/application/config/database.php

### 其他配置
* 支出类型：fm/application/config/app.php
* 站点base_url:  fm/application/config/config.php => $config['base_url'];

## 功能

* 用户管理
* 收入与支出管理
* 财务报表
